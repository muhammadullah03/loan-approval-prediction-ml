# Loan Approval Prediction Using Machine Learning

## Project Overview

This project is a supervised machine learning project where I built a model to predict whether a loan application is likely to be approved or not.

The goal of this project was to practice the full machine learning workflow, including data cleaning, exploratory data analysis, feature preparation, model training, and model evaluation.

Loan approval prediction is a useful business problem because banks and financial institutions need to evaluate applicants based on financial and personal information. A machine learning model can help identify patterns that are connected to loan approval decisions.

## Business Problem

Loan applications often include information such as income, credit history, employment status, loan amount, and other applicant details.

The main question for this project was:

**Can we use applicant information to predict whether a loan will be approved?**

This type of model could help financial institutions:

- Understand important factors in loan approval
- Support faster application screening
- Identify risk patterns
- Improve decision-making with data
- Analyze trends in approved and rejected applications

## Dataset

This project uses a loan approval dataset with applicant information and loan approval status.

The target variable is the loan approval outcome.
